Multi-Channel Test

Description

This page connects to a single different Server Sent Event (SSE) stream, listening on events named "Event1", "Event2", "Event3", and "Event4". Each separate kind of event should swap into a different container.

Example HTML

<div hx-ext="sse" sse-connect="http://localhost/posts.html?types=Event1%2cEvent2%2cEvent3%2cEvent4">
	<div sse-swap="Event1">Waiting for Posts in Event1 channel...</div>
</div>
